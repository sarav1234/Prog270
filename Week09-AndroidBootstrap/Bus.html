Android OneBusAway Application

OneBusAway is a bus tracking application. It allows you to pick a bus stop and see how long it will be before your bus arrives. It also has other features such as setting up a reminder so that you don't miss your bus.

OneBusAway Application. Bus Stop Map.

This screen uses the device's GPS location to show a map of the area you're in. It displays the various bus stops near your location. Simply touch the bus stop on the device to select it and the location displays at the top of the page.

OneBusAway Application. Bus Stop Arrivals.

With the bus stop selected, clicking on [Arrivals] displays the various buses on the way by arrival time. There is also a setting where you can filter based on the bus number. This is very useful for transit centers or downtown Seattle bus stops. In addition, it displays in how many minutes the bus will arrival along with the scheduled arrival time. When the bus is late, the time is displayed in red.

OneBusAway Application. Bus Routes.

With the bus stop selected, clicking on [Routes] will displays all the bus numbers that come to the selected bus stop. This is useful when you are not sure which stop your bus comes to as you can just click the different stops and the route are updated on the same page.
